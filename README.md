# 🇺🇸 English

## 📌 About the Project

**Personal Blog** is a web application built with **ASP.NET Core MVC** that allows users to create, read, update, and delete blog articles using a JSON file as storage.

This project simulates a real blog system with full CRUD functionality and MVC architecture.

---

## 🎯 Project Reference

This project was built based on:

https://roadmap.sh/projects/personal-blog

---

## 🧠 Study Mode

This project was developed using **ChatGPT Study Mode**, where the application was built step by step with explanations and guidance instead of receiving ready-made code.

The goal was to practice:

* ASP.NET MVC
* Controllers and Views
* Services
* JSON file manipulation
* CRUD operations
* Project structure
* Dependency Injection

---

## 🚀 Features

* Create Articles
* View Articles
* Article Details
* Edit Articles
* Delete Articles
* JSON Database
* MVC Architecture
* Bootstrap Interface
